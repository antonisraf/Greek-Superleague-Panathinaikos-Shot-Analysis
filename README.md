# Panathinaikos FC Shot Analytics ☘️

## Overview
A football analytics project focused on scraping, processing, and visualizing shot data for Panathinaikos FC. This repository tracks the relationship between Expected Goals (xG) and actual scoring performance to evaluate finishing quality.

## Project Structure
The repository is organized into a modular structure for better maintainability:
- **`scripts/`**: Contains the web scraper and the logic for generating plots, including the filtering of blocked shots.
- **`data/`**: Stores the raw and processed CSV data generated by the scraper.
- **`plots/`**: Stores the visual outputs (Shot Maps and Efficiency Charts) produced by the scripts.

## Key Features & Logic
- **Automated Scraper:** Collects match-specific shot data including coordinates, xG values, and outcomes.
- **Data Filtering:** Specifically excludes `blocked shots` within the scripts to ensure a more accurate representation of shot conversion and xG efficiency.
- **Advanced Visualization:** - **Shot Maps:** Utilizing `mplsoccer` to plot exact shot locations (X, Y) on a pitch overlay.
- **Efficiency Metrics:** Comparative Bar Charts for Actual vs. Expected goals.

## Tech Stack
- **Language:** Python 3.12.6
- **Libraries:** Pandas, Matplotlib, NumPy, mplsoccer
